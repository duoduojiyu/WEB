### 步骤2：创建第一个技能 - AI学习指南

**技能名称**：`ai-learning-guide`

**核心功能设计**：
```
1. 快速开始（4步流程）
   - 评估现状
   - 制定计划
   - 推荐资源
   - 实践建议

2. 学习流程框架
   - 阶段一：明确目标（学习目的、时间、基础）
   - 阶段二：制定学习路线（4个方向×4阶段）
   - 阶段三：知识整理方法（笔记模板+复习周期）
   - 阶段四：实践项目建议（初/中/高级）

3. 学习资源推荐
   - 官方文档（OpenAI、Claude、LangChain）
   - 实践平台（GitHub Codespaces、Google Colab）
   - 社区资源（Awesome列表、Hugging Face）

4. 进度跟踪建议
   - 学习日志模板
```

### 步骤3：创建第二个技能 - 资料检索与核实

用户提出新需求：
> "我想增加关于资料检索与资料核实确认非AI胡编乱造的技能"

**技能名称**：`research-verify`

**核心功能设计**：
```
1. 资料检索流程
   - 明确检索目标
   - 选择检索渠道（按优先级分级）
   - 搜索技巧（精确搜索、AI领域专用搜索）

2. 可信度评估框架
   - A级：官方文档（95%+可信度）
   - B级：知名专家博客（80-95%）
   - C级：社区文章（60-80%）
   - D级：无来源信息（<60%）

3. AI幻觉识别与核实
   - 5种常见幻觉类型
   - 核实流程（标记→查官方→交叉验证）
   - 快速核实技巧（论文/代码/数据/人物）

4. 代码验证详解（新增）
   - 6类常见AI代码幻觉
   - 6步代码验证流程
   - 各语言验证要点（Python/JS/Java/Go/Rust）
   - API验证清单
   - 依赖版本验证
```

## 五、导出与上传说明

### 5.1 导出对话记录的方法

#### 方法一：/export 命令（推荐）
```bash
# 在对话输入框输入
/export

# 或指定格式
/export --format markdown
```

**步骤**：
1. 输入 `/export` 命令
2. 系统自动下载 `.md` 文件到浏览器下载目录
3. 文件名格式：`conversation_2026-04-08_xx-xx-xx.md`

#### 方法二：手动复制
1. 全选对话内容（Ctrl+A）
2. 复制（Ctrl+C）
3. 粘贴到记事本/Word（Ctrl+V）
4. 保存为 `.md` 或 `.docx` 文件

---

### 5.2 上传到 GitHub 的方法

#### 方法一：网页直接上传（最简单，推荐）

**步骤1：进入仓库**
- 打开 https://github.com/duoduojiyu/WEB

**步骤2：上传文件**
1. 点击页面中间的 **Add file** 按钮
2. 选择 **Upload files**

**步骤3：选择文件**
1. 点击 **choose your files** 或拖拽文件到虚线框
2. 选择以下4个文件：
   - `ai-learning-guide.md`
   - `research-verify.md`
   - `使用说明.md`
   - `完整对话记录.md`

**步骤4：填写提交信息**
1. 在 **Commit changes** 下方填写：
   - 第一个框（标题）：`添加AI学习技能文档`
   - 第二个框（描述，可选）：`包含ai-learning-guide和research-verify两个技能`

**步骤5：提交**
- 点击绿色按钮 **Commit changes**
- 等待上传完成

**步骤6：验证**
- 刷新页面，查看文件是否出现在列表中
- 点击文件名，查看内容是否正确显示

---

#### 方法二：创建新文件夹（可选）

如果想把文件放在文件夹里：

1. 点击 **Add file** → **Create new file**
2. 在文件名框输入：`skills/ai-learning-guide.md`
   - （输入 `/` 自动创建文件夹）
3. 把文件内容复制粘贴到编辑框
4. 点击 **Commit new file**
5. 重复上述步骤上传其他文件
