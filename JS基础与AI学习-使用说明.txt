🎓 大学生如何使用这套资料
===============================================

【场景1】系统学习JS基础
--------------------------------
1. 先阅读 "JS基础与AI学习指南.md"
2. 按顺序学习：变量→函数→表达式→异步
3. 完成练习题检验
4. 用AI辅助理解难点

【场景2】使用Claude Code学习
--------------------------------
方法A：直接提问
- 打开Claude Code
- 输入问题，例如：
  "用通俗易懂的话解释闭包，给3个例子"

方法B：使用Skill模板
- 打开 js-learning-skill/SKILL.md
- 查看 prompts/ 目录下的模板
- 按模板格式提问，获得更系统的回答

方法C：边学边练
- 看cheatsheets/速查表学习知识点
- 做exercises/练习题
- 不会的用AI讲解

【场景3】做课程作业/项目
--------------------------------
1. 打开 prompts/project.md
2. 使用"项目需求分析"模板
3. 按步骤让AI指导完成
4. 最后做代码Review

【场景4】调试代码
--------------------------------
1. 打开 prompts/debug.md
2. 使用"基础报错排查"模板
3. 粘贴代码和错误信息
4. 根据AI分析修改

===============================================
💡 学习建议
===============================================

1. 循序渐进
   - 第1周：变量、函数基础
   - 第2周：DOM操作、事件
   - 第3周：异步编程
   - 第4周：做小项目

2. 善用AI但不依赖
   - 先自己思考
   - 再和AI讨论
   - 最后自己实现一遍

3. 多动手实践
   - 看完知识点写代码
   - 做完练习题再学新的
   - 每周至少完成1个小项目

4. 建立知识体系
   - 用思维导图整理
   - 定期复习
   - 教别人（最好的学习方式）
