## 📤 如何导出对话内容（上传到学习通）

### 方法一：使用 /export 命令（推荐）

#### 步骤1：执行导出命令
在对话输入框中输入以下命令：

```bash
/export
```

或导出特定格式：
```bash
/export --format markdown
/export --format json
/export --format txt
```

#### 步骤2：保存导出文件
- 执行命令后，系统会生成一个包含完整对话内容的文件
- 文件通常会自动下载到浏览器的默认下载目录
- 文件名格式：`conversation_YYYY-MM-DD_HH-MM-SS.md`

#### 步骤3：上传到学习通
1. 登录学习通网站或APP
2. 进入对应课程页面
3. 找到作业/资料上传入口
4. 选择导出的 `.md` 或 `.txt` 文件上传
5. 如有需要，可转换为 `.docx` 或 `.pdf` 格式再上传

---

### 方法二：手动复制粘贴

#### 步骤1：复制对话内容
1. 在对话界面，从第一条消息开始选择
2. 按住鼠标拖动选择全部对话内容
3. 按 `Ctrl+C` 复制（Mac用 `Cmd+C`）

#### 步骤2：粘贴到文档
1. 打开 Word / WPS / Notepad
2. 按 `Ctrl+V` 粘贴内容
3. 调整格式，添加标题和分段

#### 步骤3：保存并上传
1. 保存为 `.docx` 或 `.pdf` 格式
2. 上传到学习通对应位置

---

### 方法三：浏览器开发者工具（高级）

#### 步骤1：打开开发者工具
- 按 `F12` 或 `Ctrl+Shift+I`（Mac用 `Cmd+Option+I`）

#### 步骤2：导出HTML
1. 在 Elements/元素 标签页找到对话容器
2. 右键点击 → Copy → Copy outerHTML
3. 粘贴到文本编辑器，保存为 `.html` 文件

#### 步骤3：转换格式
- 使用在线工具将 HTML 转换为 Word/PDF
- 或直接上传 HTML 文件（如果学习通支持）

## 🔧 常见问题

### Q1: /export 命令无法使用怎么办？
**A**: 尝试以下方法：
- 检查是否在支持的平台上使用
- 尝试手动复制粘贴方法
- 联系平台管理员确认功能是否开启

### Q2: 导出文件格式不支持上传？
**A**: 
- 使用格式转换工具（如 Pandoc、在线转换器）
- 将 `.md` 转换为 `.docx` 或 `.pdf`
- 或直接复制内容粘贴到 Word 中保存

### Q3: 对话太长，导出失败？
**A**:
- 分段导出：每次只导出部分内容
- 使用浏览器开发者工具手动提取
- 或分段复制粘贴到文档中

### Q4: 学习通上传文件大小限制？
**A**:
- 压缩文件（使用ZIP格式）
- 或分割成多个小文件上传
- 或将内容粘贴到在线文档，分享链接
